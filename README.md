Java Project:
This is a Java-based application developed to demonstrate core Java programming concepts and practical problem-solving skills. The project follows a modular structure and includes backend logic with proper use of classes, methods, and object-oriented principles.

Features:
- User-friendly functionality
- Efficient data handling
- Modular and maintainable code structure
- Implementation of core Java concepts

Technologies Used:
- Java
- JDBC 
- MySQL 
- NetBeans 

Purpose:
The main objective of this project is to apply Java programming concepts in a real-world application and strengthen backend development skills.

How to Run:
1. Open the project in a NetBeans IDE
2. Configure database settings 
3. Compile and run the main Java file
